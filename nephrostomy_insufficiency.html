Несостоятельность нефростомы

Нефункционирующая нефростома · T82.8
Нефростома слева · Отсутствие оттока мочи по дренажу
📍 Локализация
В проекции левой поясничной области (анатомическая зона расположения левой почки) определяется чрескожный нефростомический дренаж.
🔍 Данные осмотра
  • Нефростомический катетер фиксирован к коже (липким пластырем / шовным материалом), положение его визуально не изменено.
  • Повязка в области стомы сухая, чистая, промокания мочой нет.
  • Кожные покровы вокруг места выхода дренажа не изменены: гиперемии, отека, инфильтрации, гнойного или серозного отделяемого не отмечается (признаки воспаления отсутствуют).
  • Послеоперационная рана (место пункции) без признаков мацерации, грануляций, налета фибрина.
🖐 Пальпаторные данные
  • Пальпация области нефростомы и окружающих мягких тканей поясничной области безболезненна.
  • Флюктуации, инфильтрации, крепитации в паранефральной области не определяется (признаков затека мочи, гематомы или абсцедирования нет).
  • При пальпации по ходу катетера болезненность отсутствует.
💧 Функциональный статус
  • Отток мочи по нефростомическому катетеру отсутствует (мочеприемник пуст, повязка сухая).
  • При попытке аспирации шприцем (или промывания) — моча не поступает, ощущается сопротивление (промывание затруднено / без эффекта).
🌡️ Дополнительные данные
  • Лихорадки, озноба нет (признаков системной воспалительной реакции не выявлено).
  • Послеоперационный рубец (если имеется) без особенностей.
🔍 Возможные причины нефункционирования дренажа
Обструкция катетера
Закупорка солями, сгустками крови, детритом
Миграция / дислокация
Смещение дренажа из лоханки почки
Перегиб катетера
Механическое препятствие оттоку
Обтурация
Закрытие отверстий катетера тканями
📋 Заключение
Клиническая картина соответствует нефункционирующей нефростоме слева (отсутствие оттока мочи по дренажу) без признаков местного инфекционно-воспалительного процесса. Кожные покровы вокруг стомы не изменены, гиперемии, отека, инфильтрации нет. Пальпация области безболезненна, признаков затека мочи или абсцедирования не выявлено. Пациент афебрилен. Требуется уточнение причины обструкции (закупорка катетера, миграция, дислокация, перегиб, обтурация солями).
