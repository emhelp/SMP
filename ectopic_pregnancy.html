📌 Код по МКБ-10: O00.9 (Внематочная беременность неуточненная) / O00.1 (Трубная беременность)
🧾 Диагноз: Основное заболевание: Прогрессирующая / нарушенная по типу разрыва маточной трубы / трубного аборта внематочная (эктопическая) беременность справа/слева.
Осложнения: Внутрибрюшное кровотечение. Геморрагический шок I–II–III степени (в зависимости от объема кровопотери). Острый болевой синдром.
Сопутствующие заболевания: Не выявлены / Хронический сальпингоофорит / Спаечная болезнь малого таза / Состояние после экстирпации матки (редко).
📢 ЖАЛОБЫ
На внезапную острую, «кинжальную» схваткообразную / постоянную боль внизу живота, преимущественно справа/слева, иррадиирующую в прямую кишку, крестец, поясницу, внутреннюю поверхность бедра (симптом «френикус» – боль в надключичной области при раздражении диафрагмального нерва кровью). Резкую слабость, головокружение, потемнение в глазах, «мушки» перед глазами, шум в ушах. Тошноту, рвоту. Кратковременную потерю сознания (синкопе). Кровянистые выделения из половых путей скудные, мажущие, темного цвета («кофейная гуща», «грязная»), на фоне задержки менструации.

Аллергологический анамнез: Не отягощен / отягощен (указать).
Менструальный анамнез: Менструальная функция с 13 лет, цикл установился сразу, по 4–5 дней, через 28 дней, регулярный, безболезненный / болезненный. Последняя менструация: (дата) – задержка менструации на 1,5–2–3 недели (или последняя менструация была необычно скудной, короткой – «псевдоменструация»).
Гинекологический анамнез: Беременности в анамнезе: (количество, роды, аборты, выкидыши, внематочная беременность – было/не было, оперативное лечение – было/не было). Гинекологические заболевания: хронический сальпингоофорит (двусторонний/правосторонний/левосторонний), спаечная болезнь малого таза, бесплодие, апоплексия яичника, кисты яичников. Операции на маточных трубах (реконструктивно-пластические), стерилизация. Контрацепция: внутриматочная спираль (ВМС) – (есть/нет, срок установки), гормональная контрацепция – (принимает/не принимает, регулярность).
Анамнез заболевания: Заболела остро, 1–2–3 часа назад / сегодня / вчера. Боль возникла на фоне полного покоя / после физической нагрузки / после полового акта / после дефекации. Сознание теряла / не теряла (указать). Самостоятельно анальгетики не принимала / принимала (указать) без эффекта.
🩺 ОБЪЕКТИВНО
Общее состояние: Тяжелое / Крайне тяжелое (зависит от объема кровопотери).
Сознание: Ясное / Оглушение / Сопор (при значительной кровопотере).
Шкала комы Глазго (GCS): 15 / 14 / 13 баллов (E3–4, V4–5, M6).
Положение: Вынужденное (лежит на боку с подтянутыми к животу коленями, «поза эмбриона»; любая перемена положения усиливает боль и слабость; при геморрагическом шоке – гиподинамия).
Кожные покровы: Резко бледные, холодные, влажные, липкий пот. Акроцианоз, цианоз губ, носогубного треугольника. Сыпи нет. Желтушности нет.

🔍 STATUS LOCALIS
Наружные половые органы: Развиты правильно, по женскому типу.
Влагалищное исследование (проводится строго при подозрении на внематочную беременность, в условиях стационара или специализированной бригадой СМП с соблюдением правил асептики):
Шейка матки: Чистая, цианотичная (признак беременности). Зев закрыт.
Выделения из половых путей: Скудные, темно-кровянистые, «мажущие», «грязные», цвета «кофейной гущи», без сгустков. Патологических (гнойных, пенистых, творожистых) выделений нет.
Своды влагалища: Уплощение и резкая болезненность заднего свода – симптом «крик Дугласа» (патогномоничный признак гемоперитонеума).
Матка: Несколько увеличена (признаки беременности), размягчена, безболезненная.
Придатки: Справа/слева от матки пальпируется резко болезненное, тестоватой консистенции образование (маточная труба с плодным яйцом). Четкие контуры не определяются из-за болезненности и напряжения мышц. Противоположные придатки без особенностей.
Psoas-симптом: Положительный на стороне поражения.

📌 Примечание для бригады СМП:
Дифференциальная диагностика проведена с:
  • апоплексией яичника (нет задержки менструации, отрицательный тест на беременность, боль возникает в середине цикла);
  • острым аппендицитом (миграция боли, симптомы Ровзинга/Ситковского положительные, нет признаков беременности);
  • перекрутом кисты яичника (нарастающая боль, тошнота, рвота, нет задержки менструации);
  • острым сальпингоофоритом (лихорадка, гнойные выделения, двухсторонняя боль);
  • гастритом, панкреатитом (связь с приемом пищи, опоясывающая боль).
Тест на беременность (моча): Экспресс-метод – положительный (обязательно выполнить при подозрении!).
