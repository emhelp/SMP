📱 Установите приложение "Скорая помощь"

Работает даже без интернета!

🤖

Android

Chrome
1
Установите Google Chrome из Play Маркет
2
Откройте сайт: https://emhelp.github.io/SMP/index.html
3
Нажмите «Установить приложение» внизу экрана или дождитесь появления баннера вверху экрана с предложением установки
⚡ ВАЖНО: Если баннер не появился — нажмите ⋮ (три точки) в правом верхнем углу → «Установить приложение»
🍎

iPhone

Safari
1
Откройте Safari (стандартный браузер)
2
Перейдите на сайт: https://emhelp.github.io/SMP/index.html
3
Нажмите «Поделиться» ⎔
4
Выберите «На экран "Домой"» → «Добавить»
✅
Готово! Иконка приложения появится на рабочем столе.
Теперь приложение работает даже без интернета!
