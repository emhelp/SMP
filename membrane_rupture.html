МКБ-10: O42.9 — Дородовый разрыв околоплодных оболочек, неуточненный (преждевременное излитие околоплодных вод)
⚠️ ТРЕБУЕТ ГОСПИТАЛИЗАЦИИ В АКУШЕРСКИЙ СТАЦИОНАР! Риск восходящей инфекции и дистресса плода!
ДИАГНОЗ
Основное заболевание: Дородовый разрыв плодных оболочек. Преждевременное излитие околоплодных вод. Безводный промежуток ______ часов.
Осложнения: Угроза восходящей инфекции (хориоамнионит – риск). Слабость родовой деятельности (риск). Дистресс плода (при длительном безводном промежутке, маловодии, инфицировании).
Сопутствующие заболевания: Беременность ______ недель. Тазовое предлежание / поперечное положение / головное предлежание (указать). Истмико-цервикальная недостаточность (в анамнезе). Многоводие. Многоплодная беременность.
ЖАЛОБЫ
Выделения:Обильные / умеренные / скудные жидкие выделения из половых путей светлого, прозрачного цвета, без запаха / с примесью крови (при отслойке) / с зеленоватым оттенком (меконий – дистресс плода).
Симптом вытекания:Ощущение «вытекания» жидкости – выделения усиливаются при движении, смене положения тела, кашле, натуживании (положительный симптом кашлевого толчка).
Характер излития:Внезапное излитие вод «струей» / «подтекание» в течение ______ часов/дней.
Родовая деятельность:Отсутствие схваток / наличие нерегулярных / регулярных схваток (при развитии родовой деятельности).
Признаки инфекции:Изменение цвета и запаха выделений (при присоединении инфекции – зеленые, желтые, с неприятным запахом). Повышение температуры тела (при хориоамнионите).
АНАМНЕЗ
Анамнез заболевания: Излитие вод произошло (дата, время: ______). Обстоятельства: в покое / после физической нагрузки / после полового акта / после травмы / на фоне полного благополучия. Количество вод: обильное («вылилось», промокла одежда) / скудное (подтекание). Цвет, запах: прозрачные, без запаха / зеленые, мутные, с запахом. Схватки: не начались / начались через ______ часов. Самостоятельно не обращалась / обращалась в женскую консультацию / приемный покой (осмотр, проведен / не проведен тест на подтекание вод, УЗИ, влагалищное исследование). Рекомендована госпитализация: согласна / отказалась (указать причину).
Эпидемиологический анамнез: Контакта с инфекционными больными нет.
Аллергологический анамнез: Не отягощен / отягощен (указать аллергию на лекарственные препараты, особенно на антибиотики).
Гинекологический и акушерский анамнез: Менструальная функция: менархе с __ лет, цикл __ дней, по __ дней, регулярный/нерегулярный. Последняя менструация: (дата) ______. Срок беременности настоящей: ______ недель (по менструации / УЗИ). Репродуктивная функция: беременностей – __, родов – __, абортов – __, выкидышей – __. Настоящая беременность: по счету ______. Состоит / не состоит на учете в женской консультации. Течение беременности: без осложнений / осложнено (указать: ИЦН, многоводие, маловодие, кольпит, бактериальный вагиноз, пиелонефрит, носительство стрептококка группы В). Преждевременные роды в анамнезе: были / не были (фактор риска). ПРЕДЛЕЖАНИЕ ПЛАЦЕНТЫ – ИСКЛЮЧЕНО! (по данным УЗИ). Гинекологические заболевания: кольпит, бактериальный вагиноз, цервицит, ИППП (в анамнезе), истмико-цервикальная недостаточность (ИЦН) – коррекция (пессарий / швы на шейку матки).
Данные лабораторного обследования (из обменной карты): Мазок на флору (срок, результат). Посев на стрептококк группы В (35–37 недель, результат). Тест на подтекание вод (амниотест, ПАМГ-1) – при наличии.
Выделения из половых путей: На момент осмотра: жидкие, обильные / умеренные / скудные, прозрачные / беловатые / зеленые / с примесью крови / с хлопьями мекония, без запаха / с неприятным запахом.
ОБЪЕКТИВНО
Состояние: удовлетворительное / средней степени тяжести
Сознание: ясное (GCS 15 баллов)
Положение: активное
Кожные покровы: Чистые, обычной окраски, влажность нормальная. Сыпи, цианоза, желтушности нет.
Температура: 36,6 °C / 37,5–38,0 °C (при хориоамнионите).
ОРГАНЫ ДЫХАНИЯ
ЧДД:16–18 в мин.
Аускультативно:Дыхание везикулярное, проводится во все отделы, хрипов нет.
ОРГАНЫ КРОВООБРАЩЕНИЯ
Пульс:72–80–90 в мин (при инфекции – тахикардия), ритмичный, удовлетворительного наполнения. ЧСС: 72–80–90 в мин.
АД:110/70 – 120/80 мм рт.ст. (привычное для пациентки).
Тоны сердца:Звучные, ритмичные.
ОРГАНЫ ПИЩЕВАРЕНИЯ
Живот:Увеличен за счет беременной матки. Мягкий, не вздут, участвует в акте дыхания, безболезненный при пальпации во всех отделах. Болезненность в гипогастрии – при хориоамнионите, угрозе преждевременных родов.
Симптомы раздражения брюшины:Отрицательные.
STATUS LOCALIS (акушерский статус)
Высота стояния дна матки (ВДМ): ______ см (соответствует сроку беременности).
Окружность живота (ОЖ): ______ см.
Положение плода: продольное / поперечное / косое.
Предлежание плода: головное / тазовое.
Сердцебиение плода: ясное, ритмичное, ______ уд/мин (норма 120–160). Признаки дистресса: тахикардия >160, брадикардия <110, глухость тонов, аритмия.
Двигательная активность плода: сохранена / снижена / отсутствует.
Симптомы дородового разрыва плодных оболочек:
• Визуализация: жидкие выделения, стекающие по бедру, пропитывающие прокладку, одежду.
• Симптом кашлевого толчка: при натуживании, кашле количество выделений увеличивается (положительный / сомнительный / отрицательный).
• Запах: отсутствие патологического запаха / неприятный, гнилостный (при хориоамнионите).
• Цвет: прозрачные / беловатые / зеленые (меконий) / желтые (инфекция) / с примесью крови (отслойка, предлежание плаценты – исключить!).
Выделения из половых путей: обильные / умеренные / скудные, жидкие, «водянистые», легко впитываются в прокладку. Цвет: прозрачные / беловатые / зеленоватые / желтые / с примесью крови. Запах: обычный / неприятный, гнилостный.
Наружные половые органы: Развиты правильно, по женскому типу. В области преддверия влагалища, на промежности – выделения.
КЛЮЧЕВЫЕ ПРИЗНАКИ ДОРОВОГО РАЗРЫВА ОКОЛОПЛОДНЫХ ОБОЛОЧЕК

• Жидкие, водянистые выделения.

• Положительный симптом кашлевого толчка (усиление выделений при кашле, натуживании).

• Усиление выделений при движении, смене положения.

• Отсутствие слизи, зуда, жжения.

• Данные анамнеза – внезапное излитие жидкости.

ОЦЕНКА РИСКА ХОРИОАМНИОНИТА
Длительность безводного промежутка: >18–24 часов – высокий риск.
Лихорадка >38,0°C: хориоамнионит.
Тахикардия плода >160 уд/мин: риск инфекции.
Гнойные, зловонные выделения: хориоамнионит.
Болезненность матки при пальпации: хориоамнионит.
ОЦЕНКА СОСТОЯНИЯ ПЛОДА

• 120–160 уд/мин – норма.

• Тахикардия >160 уд/мин – начальная гипоксия, риск инфекции.

• Брадикардия <110 уд/мин – тяжелая гипоксия.

• Отсутствие сердцебиения – антенатальная гибель плода.

ДИФФЕРЕНЦИАЛЬНАЯ ДИАГНОСТИКА И ТАКТИКА
Дифференциальная диагностика проведена с:
— Недержанием мочи: выделения усиливаются после мочеиспускания, отрицательный симптом кашлевого толчка, моча имеет цвет и запах, может быть стрессовым (чихание, смех).
— Обильными белями (кольпит, бактериальный вагиноз): густые, слизистые, творожистые, пенистые выделения, не усиливаются при кашле, сопровождаются зудом, жжением, pH влагалищного отделяемого – кислая среда (воды – щелочная).
— Отхождением слизистой пробки: густые, слизистые, желеобразные выделения, возможно с прожилками крови, однократное отхождение, отсутствие постоянного «подтекания».
— Предлежанием плаценты: кровянистые выделения, чаще безболезненные. Абсолютное противопоказание к влагалищному исследованию до стационара!

Примечание для бригады СМП:
— Экстренная госпитализация в акушерский стационар (родильный зал, отделение патологии беременных).
— Транспортировка на носилках в положении лежа на спине (при тазовом предлежании, поперечном положении – строго лежа).
— Категорически запрещены влагалищное исследование, пальпация шейки матки, введение тампонов, проведение клизм!
— При подозрении на предлежание плаценты (кровянистые выделения) – транспортировка с приподнятым ножным концом, исключить любые манипуляции во влагалище.
— При признаках хориоамнионита – антибактериальная терапия в стационаре.
— Мониторинг сердцебиения плода каждые 15–30 минут.
